# Moon Rise Simulator

이 프로그램은 사용자가 지정한 장소(위도, 경도)와 날짜에 달이 뜨는 시간과 방위각을 계산합니다.

## 설치
1. Python 3.x 설치.
2. `pip install -r requirements.txt` 실행 (Skyfield 라이브러리 설치).

## 사용법
1. `python moon.py` 실행.
2. 위도, 경도, 날짜 입력.

## 예시
- 위도: 37.5665 (서울)
- 경도: 126.9780
- 날짜: 2024-10-01

출력: 뜨는 시간과 방위각.

## 주의
